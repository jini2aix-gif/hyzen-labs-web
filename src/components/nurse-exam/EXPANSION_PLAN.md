# 간호조무사 1000문항 데이터베이스 확장 프로젝트

## 현황
- 현재: 105문항 (기초35, 보건15, 공중20, 실기35)
- 목표: 1000문항

## 확장 계획 (895문항 추가)

### Phase 1: 기초간호학 확장 (298문항 추가, 총 333문항)
#### 세부 주제별 배분:
1. 해부생리학 (80문항)
   - 순환계, 호흡계, 소화계, 비뇨계, 신경계, 내분비계, 근골격계, 감각기계
2. 약리학 (60문항)
   - 약물 관리, 투약 경로, 약물 작용, 부작용, 주요 약물별 특성
3. 기본간호학 (80문항)
   - 활력징후, 체위, 개인위생, 배설, 영양, 산소요법, 상처관리
4. 성인간호학 (40문항)
   - 만성질환, 수술간호, 응급간호
5. 모성간호학 (20문항)
   - 임신, 분만, 산욕기 간호
6. 아동간호학 (18문항)
   - 성장발달, 예방접종, 소아질환

### Phase 2: 보건간호학 확장 (128문항 추가, 총 143문항)
1. 지역사회간호 (50문항)
2. 모자보건 (30문항)
3. 학교보건 (25문항)
4. 산업보건 (18문항)

### Phase 3: 공중보건학 확장 (170문항 추가, 총 190문항)
1. 감염병 관리 (80문항)
   - 법정감염병, 예방접종, 소독/멸균
2. 환경보건 (50문항)
   - 대기/수질/토양오염, 폐기물 관리
3. 보건행정 (40문항)
   - 보건정책, 의료법규, 건강보험

### Phase 4: 실기학 확장 (299문항 추가, 총 334문항)
1. 기본간호술기 (100문항)
   - 활력징후, 체위변경, 개인위생, 배설간호
2. 투약 및 주사 (80문항)
   - 경구투약, 근육주사, 피하주사, 정맥주사
3. 무균술 및 감염관리 (60문항)
   - 손씻기, 격리, 멸균장갑, 소독
4. 응급처치 (50문항)
   - CPR, 기도폐쇄, 출혈, 쇼크, 골절
5. 특수간호술기 (44문항)
   - 흡인, 도뇨, 관장, 산소요법, 상처드레싱
